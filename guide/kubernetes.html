Kubernetes Integration¶

The easiest way to scale out using Scanner is via kubernetes. Kubernetes is tool for automating the deployment and management of a cluster of machines that run containers (a container is akin to a light-weight VM image).

We provide a step-by-step example of getting Scanner up and running on your own kubernetes cluster using either Google Cloud Platform or Amazon Web Services.
